PART 2: KEY CONCEPTS YOU NEED TO UNDERSTAND
================================================================================

Concept 1: Understanding the Puzzle Format
-------------------------------------------
The puzzle is a string where:
- NUMBERS = starting positions of words (indicates how many words start there)
- DOTS (.) = blocked spaces that can't be filled
- NEWLINES (\n) = row separators

Think of numbers as "word start markers" on a game board. The number itself tells 
you how many words begin at that position (usually 1 for just horizontal, 
1 for just vertical, or 2 for both).

Example puzzle breakdown:
    2001
    0..0
    1000
    0..0

Here:
- Position (0,0) has "2" = 2 words start here (1 horizontal + 1 vertical)
- Position (0,1) has "0" = no words start here
- The "." means blocked spaces
- "0" means empty space where we might place part of a word (not a start)

Concept 2: Horizontal vs Vertical Words
----------------------------------------
In crosswords, words run two directions:
- HORIZONTAL: Left to right (→)
- VERTICAL: Top to bottom (↓)

Think of it like reading:
- Horizontal words are like reading a line in a book
- Vertical words are like reading the side of a bookmark

When we place a word, we need to check:
1. Does it fit without hitting a dot or going out of bounds?
2. Does it intersect correctly with other words already placed?
3. Is the letter the same where words cross?

Concept 3: Backtracking - The Trial and Error Method
----------------------------------------------------
Backtracking is like trying different combinations until you find the right one.

Imagine you're in a maze:
1. You try going left
2. You hit a dead end, so you BACKTRACK and go right
3. You repeat this process until you find the exit

In our crossword:
1. Try placing word #1 in position #1
2. Try placing word #2 - if it doesn't fit, backtrack
3. Go back and try word #1 in a different position
4. Keep trying until you solve it (or run out of options)

Concept 4: Clues, Clue Numbers, and Directions
----------------------------------------------
In a real crossword, you see: "1 Across: Four-legged animal" or "1 Down: Opposite of up"

In our puzzle:
- A NUMBER indicates a clue position
- Words starting from that position are either:
  - ACROSS (Horizontal)
  - DOWN (Vertical)

When you see a number with value "2", it means:
- 1 word goes horizontally from this point
- 1 word goes vertically from this point
- Total = 2 words start here

Concept 5: Uniqueness Constraint
--------------------------------
Here's a crucial rule: The puzzle must have exactly ONE solution.

Why? Because a crossword is only fun if there's a definite answer!

If we find:
- 0 solutions → Error (puzzle is impossible)
- 1 solution → Success! (print the solved puzzle)
- 2+ solutions → Error (puzzle is ambiguous)

PART 3: STEP-BY-STEP ALGORITHM (Your Thinking Process)
================================================================================

Step 1: Parse the Puzzle
   - Convert the puzzle string into a 2D grid (array of arrays)
   - Identify all starting positions (numbers) and their counts
   - Identify all words that need to fit (horizontal and vertical)

Step 2: Generate Word Slots
   - For each starting position, find out:
     * What words can go horizontally from here (and how long)
     * What words can go vertically from here (and how long)
   - Create a list of "slots" that need to be filled
   
Step 3: Validate Input
   - Check if the puzzle and word list are valid
   - Make sure all words are used exactly once (no duplicates)
   - Verify that the puzzle follows the rules
   
Step 4: Use Backtracking to Try All Combinations
   - For each slot, try placing each available word
   - Check if it fits:
     * Doesn't hit blocked spaces (dots)
     * Doesn't go out of bounds
     * Matches letters where it intersects with other words
   - If a word works, mark it as used and move to next slot
   - If no word works, backtrack and try a different word
   
Step 5: Count Solutions
   - Keep track of how many valid complete solutions you find
   - Stop searching after finding 2 solutions (optimization)
   
Step 6: Output
   - If exactly 1 solution found: Print the solved puzzle
   - Otherwise: Print 'Error'

// Main function that solves the crossword puzzle
function crosswordSolver(puzzle, words) {
  // First, parse the puzzle string into a 2D grid
  const grid = puzzle.split('\n').map(row => row.split(''));
  const height = grid.length;
  const width = grid[0].length;

  // Find all starting positions and create word slots
  const slots = [];
  
  for (let row = 0; row < height; row++) {
    for (let col = 0; col < width; col++) {
      const cell = grid[row][col];
      
      // Check if this cell is a starting position (has a number)
      if (cell !== '.' && cell !== '0' && isNaN(cell) === false && cell !== '\n') {
        // This is a starting position!
        
        // Check for a horizontal word starting here
        if (col === 0 || grid[row][col - 1] === '.') {
          // Find the length of this horizontal word
          let len = 0;
          let c = col;
          while (c < width && grid[row][c] !== '.') {
            len++;
            c++;
          }
          if (len > 1) { // Only add if word is more than 1 letter
            slots.push({ row, col, horizontal: true, length: len });
          }
        }
        
        // Check for a vertical word starting here
        if (row === 0 || grid[row - 1][col] === '.') {
          // Find the length of this vertical word
          let len = 0;
          let r = row;
          while (r < height && grid[r][col] !== '.') {
            len++;
            r++;
          }
          if (len > 1) { // Only add if word is more than 1 letter
            slots.push({ row, col, horizontal: false, length: len });
          }
        }
      }
    }
  }

  // Validate input
  if (words.length !== slots.length) {
    return 'Error'; // Number of words doesn't match number of slots
  }

  // Use backtracking to find solution
  const used = new Array(words.length).fill(false);
  let solutionCount = 0;
  let solution = null;

  function backtrack(slotIndex) {
    // If we've placed all words, we found a solution!
    if (slotIndex === slots.length) {
      solutionCount++;
      if (solutionCount === 1) {
        // Save the first solution
        solution = grid.map(row => [...row]);
      }
      return solutionCount <= 1; // Stop after finding 2 solutions
    }

    const slot = slots[slotIndex];
    const wordLength = slot.length;

    // Try each word that hasn't been used yet
    for (let i = 0; i < words.length; i++) {
      if (used[i]) continue; // Skip already-used words
      
      const word = words[i];
      
      // Check if word length matches slot length
      if (word.length !== wordLength) continue;
      
      // Try to place the word in this slot
      if (canPlaceWord(grid, word, slot)) {
        // Place the word
        placeWord(grid, word, slot);
        used[i] = true;

        // Recursively try to place the next word
        if (!backtrack(slotIndex + 1)) {
          // We found 2+ solutions, stop searching
          removeWord(grid, word, slot);
          used[i] = false;
          return false;
        }

        // Remove the word (backtrack)
        removeWord(grid, word, slot);
        used[i] = false;
      }
    }

    return true; // Continue searching
  }

  // Helper function: Check if a word can be placed in a slot
  function canPlaceWord(grid, word, slot) {
    const { row, col, horizontal, length } = slot;

    for (let i = 0; i < length; i++) {
      let r = row;
      let c = col;
      
      if (horizontal) {
        c += i;
      } else {
        r += i;
      }

      // Check bounds
      if (r >= height || c >= width) return false;

      const cell = grid[r][c];
      
      // Cell must be empty or match the word's letter
      if (cell !== '0' && cell !== word[i]) {
        return false;
      }
    }

    return true;
  }

  // Helper function: Place a word on the grid
  function placeWord(grid, word, slot) {
    const { row, col, horizontal, length } = slot;

    for (let i = 0; i < length; i++) {
      let r = row;
      let c = col;
      
      if (horizontal) {
        c += i;
      } else {
        r += i;
      }

      grid[r][c] = word[i];
    }
  }

  // Helper function: Remove a word from the grid
  function removeWord(grid, word, slot) {
    const { row, col, horizontal, length } = slot;

    for (let i = 0; i < length; i++) {
      let r = row;
      let c = col;
      
      if (horizontal) {
        c += i;
      } else {
        r += i;
      }

      grid[r][c] = '0'; // Reset to empty marker
    }
  }

  // Start the backtracking algorithm
  backtrack(0);

  // Return result based on solution count
  if (solutionCount === 1) {
    const result = solution.map(row => row.join('')).join('\n');
    console.log(result);
    return result;
  } else {
    console.log('Error');
    return 'Error';
  }
}

Let's trace through the example step by step:

Initial grid after parsing:
  Position (0,0): '2' means 2 words start here
  Position (0,1): '0' means empty cell
  Position (0,2): '0' means empty cell
  Position (0,3): '1' means 1 word starts here (vertical only)

Words available: ['casa', 'alan', 'ciao', 'anta']

Slot identification:
  Slot 1: row=0, col=0, horizontal=true, length=4   → needs "casa"
  Slot 2: row=0, col=0, horizontal=false, length=4  → needs "ciao"
  Slot 3: row=0, col=3, horizontal=false, length=4  → needs "alan"
  Slot 4: row=2, col=0, horizontal=true, length=4   → needs "anta"

Backtracking process:
  
  Attempt 1: Slot 1 (horizontal, length 4)
    - Try "casa": Can place? Yes!
    - Grid becomes: ['c','a','s','a'], ['.','.','.'], etc.
    
    Attempt 2: Slot 2 (vertical, length 4)
      - Try "alan": Can place at (0,0)? No! "alan"[0]='a' but grid[0][0]='c'
      - Try "ciao": Can place at (0,0)? Yes! 'c' matches 'c'
      - Grid now has 'c','i','a','o' going down from (0,0)
      
      Attempt 3: Slot 3 (vertical, length 4)
        - Try "alan": Can place at (0,3)? Yes! 'a' matches 'a'
        - Grid now has 'a','l','a','n' going down from (0,3)
        
        Attempt 4: Slot 4 (horizontal, length 4)
          - Try "anta": Can place at (2,0)? Yes! 'a' matches 'a'
          - All words placed! solutionCount = 1
          
          Continue searching to see if there's another solution...
          (After checking all possibilities, no other solution found)
          
Final result: Print the solved grid

Why this algorithm works:

✓ Backtracking explores all possible combinations systematically
✓ Early termination when letters don't match prevents invalid placements
✓ Tracking "used" words prevents using the same word twice
✓ Counting solutions ensures uniqueness
✓ The algorithm is efficient enough for typical crossword sizes

PART 8: COMMON MISTAKES TO AVOID
================================================================================

❌ Mistake 1: Not checking if words fit without hitting dots or boundaries
   Problem: Your function places words outside the grid or on blocked spaces,
            causing the puzzle to be invalid.
   Solution: In the canPlaceWord() function, always verify:
            - r >= height && c >= width (not out of bounds)
            - grid[r][c] === '.' (not a blocked space)

❌ Mistake 2: Using the same word twice
   Problem: The algorithm doesn't track which words have been used, so it tries
            to place "casa" twice.
   Solution: Use a "used" array to track which words have been placed:
            const used = new Array(words.length).fill(false);
            Mark each word as used when you place it.

❌ Mistake 3: Not correctly resetting the grid during backtracking
   Problem: When you backtrack, you leave letters on the grid from failed 
            attempts, causing incorrect matches later.
   Solution: Always call removeWord() to reset grid cells back to '0' after
            backtracking.

❌ Mistake 4: Forgetting to check if the solution is unique
   Problem: You return the first solution found without checking if there are
            others, but the puzzle might have multiple solutions.
   Solution: Keep searching after finding the first solution. Stop at 2 solutions.
            Only print if solutionCount === 1.

❌ Mistake 5: Not parsing the puzzle correctly
   Problem: You don't correctly identify starting positions (numbers) and 
            determine word slot lengths and directions.
   Solution: For each number in the grid:
            - Check horizontal: is there space to the right and no wall?
            - Check vertical: is there space below and no wall?
            - Calculate exact length by counting until you hit a dot or edge.

❌ Mistake 6: Mixing up rows and columns
   Problem: You access grid[col][row] instead of grid[row][col], causing
            the algorithm to check the wrong cells.
   Solution: Always remember: grid[row][col]
            - row comes first (vertical position)
            - col comes second (horizontal position)

```javascript
function crosswordSolver(puzzle, words) {
  // Parse the puzzle into a 2D grid
  const grid = puzzle.split('\n').map(row => row.split(''));
  const height = grid.length;
  const width = grid[0].length;

  // Find all word slots (horizontal and vertical)
  const slots = [];
  
  for (let row = 0; row < height; row++) {
    for (let col = 0; col < width; col++) {
      const cell = grid[row][col];
      
      // Check if this is a starting position (a number)
      if (cell !== '.' && cell !== '0' && !isNaN(cell)) {
        // Check for horizontal word
        if (col === 0 || grid[row][col - 1] === '.') {
          let length = 0;
          let c = col;
          while (c < width && grid[row][c] !== '.') {
            length++;
            c++;
          }
          if (length > 1) {
            slots.push({ row, col, horizontal: true, length });
          }
        }
        
        // Check for vertical word
        if (row === 0 || grid[row - 1][col] === '.') {
          let length = 0;
          let r = row;
          while (r < height && grid[r][col] !== '.') {
            length++;
            r++;
          }
          if (length > 1) {
            slots.push({ row, col, horizontal: false, length });
          }
        }
      }
    }
  }

  // Validate: correct number of words for slots
  if (words.length !== slots.length) {
    console.log('Error');
    return 'Error';
  }

  // Backtracking variables
  const used = new Array(words.length).fill(false);
  let solutionCount = 0;
  let solution = null;

  function backtrack(slotIndex) {
    // All slots filled - we found a solution!
    if (slotIndex === slots.length) {
      solutionCount++;
      if (solutionCount === 1) {
        solution = grid.map(row => [...row]);
      }
      return solutionCount <= 1; // Stop after finding 2 solutions
    }

    const slot = slots[slotIndex];
    
    // Try each unused word
    for (let i = 0; i < words.length; i++) {
      if (used[i] || words[i].length !== slot.length) continue;

      const word = words[i];
      
      // Check if word fits
      if (canPlace(word, slot)) {
        // Place word
        place(word, slot);
        used[i] = true;
        
        // Continue backtracking
        if (!backtrack(slotIndex + 1)) {
          // Found 2+ solutions, stop
          remove(word, slot);
          used[i] = false;
          return false;
        }
        
        // Backtrack
        remove(word, slot);
        used[i] = false;
      }
    }
    
    return true;
  }

  function canPlace(word, slot) {
    const { row, col, horizontal, length } = slot;
    
    for (let i = 0; i < length; i++) {
      const r = horizontal ? row : row + i;
      const c = horizontal ? col + i : col;
      
      if (r >= height || c >= width) return false;
      
      const cell = grid[r][c];
      if (cell !== '0' && cell !== word[i]) return false;
    }
    
    return true;
  }

  function place(word, slot) {
    const { row, col, horizontal, length } = slot;
    
    for (let i = 0; i < length; i++) {
      const r = horizontal ? row : row + i;
      const c = horizontal ? col + i : col;
      grid[r][c] = word[i];
    }
  }

  function remove(word, slot) {
    const { row, col, horizontal, length } = slot;
    
    for (let i = 0; i < length; i++) {
      const r = horizontal ? row : row + i;
      const c = horizontal ? col + i : col;
      grid[r][c] = '0';
    }
  }

  // Find all solutions
  backtrack(0);

  // Output based on solution count
  if (solutionCount === 1) {
    const result = solution.map(row => row.join('')).join('\n');
    console.log(result);
    return result;
  } else {
    console.log('Error');
    return 'Error';
  }
}
```
